What is DOM used for?

DOM is used to dynamically manipulate HTML elements, enabling developers to create interactive web pages by modifying structure, content, and styles.

How to create new elements?

You can create new elements using document.createElement() and append them to the DOM using methods like appendChild() or insertBefore().

How to remove DOM elements?

You can remove DOM elements using element.remove() or parentElement.removeChild(childElement) to delete specific elements from the document structure.

How to change an attribute?

You can change an element's attribute using setAttribute() or directly access it as a property, like element.src = "newURL".
